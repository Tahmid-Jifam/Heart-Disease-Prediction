# Heart Disease Prediction UI

A responsive front-end app to estimate heart disease risk based on user health inputs. Built with HTML, CSS, JavaScript, and Chart.js.

## Features
- Input fields for age, gender, blood pressure, cholesterol, blood sugar, heart rate, and chest pain type.
- Calculates risk percentage based on simple thresholds.
- Displays risk factors and results.
- Shows a pie chart visualization.
- Mobile responsive design.

## How to Use
1. Open `index.html` in any modern web browser.
2. Fill out the health parameters in the form.
3. Click "Check Risk" to see your heart disease risk estimate and chart.

## Disclaimer
This is a demonstration project and not a medical diagnosis tool.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js
